# Setting up talos on AWS with terraform

Follow [README.md](../README.md) instructions to setup brew and install the `awscli` and `terraform` packages. Rough approach assuming you have an aws account is below, getting `~/.aws/config` setup for your environment is left as an exercise for the reader.

```bash
# in taloscluster dir
aws sso configure
export AWS_PROFILE=...
export AWS_REGION=ap-southeast-2
aws sso login
terraform init
terraform plan
terraform apply
mkdir ~/.kube ~/.talos
terraform output -raw kubeconfig > ~/.kube/config
terraform output -raw talosconfig > ~/.talos/config
# Fiddle with cluster, work on storage proxies etc later
terraform destroy # This will clean up all the workshops resources
```

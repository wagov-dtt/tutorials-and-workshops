# Dev Container

A ready-to-use development environment for this repo.

## What is here

- `devcontainer.json` - VS Code / Dev Containers configuration.

## Use it

1. Install Docker and the Dev Containers extension.
2. Open this repo in VS Code.
3. Choose **Dev Containers: Reopen in Container**.
4. Run `just prereqs` inside the container if tools are missing.

This is for local development only; it does not create cloud resources.
